-------------------------INSTRUCTIONS---------------------

1. Compile the C program Lab2.c by using following command
    gcc programName.c -o executableFileName

    In practice:

    gcc Lab2.c -o lab2

2. Run the C program by using the following command:
    ./executableFileName

    In practice:

    ./lab2

3. The program will start by asking to enter the text file that contains prime numbers. You have 2 options
    a. Enter the text file name that I have already in the file (primes.txt)
    ---------------------------OR--------------------------------------
    b. Enter your own text file that has the list of numbers [2,N]. 
       Make sure to include the .txt extension for your text file name
